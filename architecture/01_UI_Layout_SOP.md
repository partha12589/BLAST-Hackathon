# 01 - UI & Dashboard Layout SOP

## Goal
Define the structure and deterministic components of the Taint & Risk Scoring Command Center dashboard based on the approved design inspiration.

## Components Mapping

1. **Left Sidebar (Navigation)**:
   - Live Mempool
   - Entity Search
   - Alerts

2. **Top Cards (Real-time Metrics)**:
   - Overall Risk Score (0-100)
   - Tainted Volume (USD/Token amount)
   - Network Hops (int)

3. **Main Bottom Section (Investigator Command Center)**:
   - Target Wallet Address
   - Deterministic Status (e.g., massive red **BLOCKED** badge)
   - Action Button: **Freeze Funds**

4. **Hero Section (Central Visualizer)**:
   - Interactive Node Graph tracking stolen money provenance (Nodes: Addresses/Users, Edges: Flow amounts).
